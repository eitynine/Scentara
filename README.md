# Scentara
Scentara hadir sebagai scent matchmaker pribadi kamu.  Kami percaya kalau parfum bukan sekadar wangi, tapi soal ekspresi diri dan aura. Menggunakan pendekatan yang cerdas dan personal, Scentara bantu kamu menemukan signature scent yang paling cocok dengan karakter, mood, hingga aktivitas harianmu.
